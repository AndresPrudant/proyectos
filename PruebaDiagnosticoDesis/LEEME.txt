###################################################################Aplicación de votación Desis###################################################################

******* ESPECIFICACIONES DE LA APLICACIÓN *******

Version PHP: 8.0.26.
Base de datos: MySQL.
Version base de datos: 8.0.31.

******* CÓMO INSTALAR *******

Para poder utilizar la aplicacion web de votación, es necesario seguir los siguientes pasos:

1- Se debe instalar MySQL en la máquina, es recomendable descargar la aplicación 'WAMP SERVER x64' ya que la base de datos fué desarrollada con esa aplicación.

2- Una vez instalado, se creará una carpeta raíz en el disco local C: llamada 'wamp64' ahí es donde se debe alojar el proyecto.

3- Descargar la aplicación PHP desde el siguiente link de github: https://github.com/AndresPrudant/proyectos.

4- Una vez descargada la aplicación, se debe crear una nueva carpeta en la siguiente ruta: C:\wamp64\www\*NuevaCarpeta* y se debe copiar la aplicación en la nueva carpeta.

5- Se debe abrir la aplicacion Wamp Server y esperar unos segundos para que los servicios se inicien. Cuando en la barra de aplicaciones aparece el logo de wamp en verde,
   ya se puede empezar a utilizar la aplicación web desde un escritorio.

6- El siguiente paso es cargar la base de datos que esta en la carpeta 'Bdd' del proyecto a WampServer, para eso es necesario abrir navegador e ingresar a la url: localhost
   una vez ingresado es necesario pinchar 'PhpMyAdmin 5.2.0' en la sección llamada 'Your Aliases'. 

7- Para ingresar a PHP MyAdmin se debe poner como nombre de usuario 'root' y contraseña dejar en vacío.

8- Una vez ingresado, para importar la BD es necesario pinchar en 'importar' donde está el menu de opciones en la parte superior, solo se debe Seleccionar el archivo
   de la bd ubicada en la carpeta 'Bdd' del proyecto, bajar hasta el final y solo pinchar en importar.

9- Como último paso, para poder utilizar la aplicación es necesario ingresar a la siguiente url desde el navegador: localhost\*NuevaCarpeta*\index.php

10- A probar!
